Writing Systems

This tool walks through a single LIFT dictionary file, reporting on Writing Systems lang tags in use. The LDML Writing System repository is checked to see if all these tags exist, with the correct file name. You can rename writing systems (and related data in the lift file), and delete unused writing systems files.

This tool can be configured under the config tab. By default the tool reports on what could be done. Use the checkboxes to enable the various fixes.

Risks

The input lift file is not modified. If the fixes are enabled the modified lift is saved as [filename].processed.lift. LDML files can be deleted by this tool, so make a backup first.
