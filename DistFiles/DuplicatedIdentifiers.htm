Duplicate Indentifiers Report

This tool walks through a single LIFT dictionary file, looking for data that has crucial information duplicated, but whihc needs to be unique. The report displays errors that hinder other programs from making use that uniquesness expectation.

Risks

No risk to the lift file itself, as this is only reads the lift file and reports on the noon-unique information it finds. Other means must then be used to select which of the duplicates needs to be removed and which can remain.

Authors

Randy Regnier. Uses the Palaso library, whose authors include Eric Albright, Tim Armstrong, John Hatton, & Cambell Prince.
