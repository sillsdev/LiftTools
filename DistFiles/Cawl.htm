Writing Systems

This tool can be used to remove unwanted glosses and definitions introduced by use of the SIL CAWL word pack in some dictionary software.

This tool can be configured under the config tab. By default the tool reports on what could be done. Use the checkboxes to enable the various fixes.

Risks

The input lift file is not modified, if the fixes are enabled the modified lift is saved as [filename].processed.lift.

Authors

Uses the Palaso library, whose authors include Eric Albright, Tim Armstrong, John Hatton, & Cambell Prince.
