Separate CAWL Senses into Entries

This tool walks through a single LIFT dictionary file, looking for entries containing multiple senses which came from different words in the CAWL word list.  When it finds them, it splits them into separate entries. The resulting LIFT file is saved with a new name.

Risks

An undiscovered bug in this tool could cause information to no be exported to the new LIFT file. The original file would be untouched.

Authors
