Orphan Files Report

This tool walks through a single LIFT dictionary file, looking for entries which look like they contain audio or images. The audio and pictures folders are then searched to see if the linked file is there or not. The report displays errors for links without files, and also files without links.

Risks

No risk to the lift file itself, as this is only read to get the names of the audio and image files.

Authors

Uses the Palaso library, whose authors include Eric Albright, Tim Armstrong, John Hatton, & Cambell Prince.
